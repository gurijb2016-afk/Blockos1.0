A Hungarian-developed Linux-based OS built from scratch.

## 📦 Structure
```
blockos/
├── boot/          # Bootloader
├── kernel/        # Kernel (process, syscall, memory)
├── drivers/       # PS/2 mouse, keyboard, VGA
├── fs/            # FAT32, RAMFS, ProcFS, VFS
├── gui/           # CFACEA GUI toolkit
├── wm/            # BlockWM compositor
├── display/       # Cursor, framebuffer
├── shell/         # BlockShell
├── network/       # Network stack
├── libc/          # Custom libc
└── install/       # Install scripts
```

## 🚀 Features
- ✅ CFACEA custom GUI
- ✅ BlockWM compositor
- ✅ FAT32 filesystem
- ✅ RAMFS + ProcFS
- ✅ PS/2 mouse driver
- ✅ Custom cursor system
- ✅ AI IDE (Claude/ChatGPT/Gemini/DeepSeek)
- ✅ Plymouth boot animation
- ✅ Calamares installer
- ✅ LibreOffice + Firefox
